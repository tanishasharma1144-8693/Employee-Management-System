# 🚀 Employee Management System

A full-stack Employee Management System (EMS) built using the MERN Stack. This application helps organizations efficiently manage employees, salaries, reports, dashboards, and administrative activities through a secure role-based system.

## 🌐 Live Demo

### Frontend
https://employee-management-system-lake-two.vercel.app

### Backend API
https://employee-management-system-z0mi.onrender.com

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/tanishasharma1144-8693/Employee-Management-System.git
```

## Install Frontend

```bash
cd client
npm install
```

## Install Backend

```bash
cd ../server
npm install
```

---

# ⚙ Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

EMAIL_USER=your_email@gmail.com

EMAIL_PASS=your_app_password
```

---

# ▶ Run Project

## Backend

```bash
cd server
npm run dev
```

## Frontend

```bash
cd client
npm run dev
```
